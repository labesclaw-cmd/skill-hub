# gmail-processor

Gmail 自動分析系統。收到 YouTube / 網頁連結信件後，自動抓取內容分析，回信摘要給用戶。

## 狀態
🚧 開發中（v0.1 規劃階段）

## 規劃功能
- 監控 Gmail 收件匣（每 5 分鐘）
- 偵測信件中的 YouTube / 網頁連結
- 自動抓取 YouTube 字幕或網頁內容
- Claude 分析重點
- 回信摘要給寄件人

## 開發階段
1. ✅ Gmail 收信監控（gmail_poller.sh 已完成）
2. ⏳ YouTube 字幕抓取
3. ⏳ 網頁內容抓取
4. ⏳ Claude 分析整合
5. ⏳ 自動回信
